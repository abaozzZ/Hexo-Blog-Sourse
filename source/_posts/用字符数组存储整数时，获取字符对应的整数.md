### 获取字符对应的整数

由于使用字符形式存储数字时，存储的是数字对应的 ASCII 码，由于在 ASCII 码表中字符‘0’到字符‘9’是连续存放的，所以下面的程序可以获得字符对应的数字：

<!-- more -->


```
char c = '2';     // c 中存储的是字符 '2' 的ASCII码
int k = c - '0';     // k 即为字符 '2' 对应的整数 2
```
